# README

## Vetores e Soma de Elementos

Este repositório contém um programa Java que demonstra a manipulação de vetores, onde dois vetores (A e B) são lidos, e um terceiro vetor (C) é gerado, contendo a soma dos elementos correspondentes de A e B. O programa solicita ao usuário a quantidade de elementos nos vetores e, em seguida, pede os valores dos vetores A e B. Ao final, imprime o vetor resultante C.

## Exemplo de Utilização

1. **Quantidade de Valores nos Vetores:**
   ```
   Quantos valores vai ter cada vetor? 6
   ```

2. **Digite os valores do vetor A:**
   ```
   8
   2
   11
   14
   13
   20
   ```

3. **Digite os valores do vetor B:**
   ```
   5
   10
   3
   1
   10
   7
   ```

4. **Vetor Resultante (C):**
   ```
   VETOR RESULTANTE:
   13
   12
   14
   15
   23
   27
   ```

## Código Fornecido

O código fornecido para esta funcionalidade está no arquivo `SomaVetores.java` no diretório correspondente. Ele utiliza um loop para ler os valores dos vetores A e B, calcula a soma dos elementos correspondentes e armazena no vetor C, e, por fim, imprime o vetor resultante.

## Conclusão

Este projeto exemplifica a manipulação de vetores em Java, demonstrando como realizar a soma de elementos correspondentes de dois vetores e armazenar o resultado em um terceiro vetor. Experimente modificar o código para explorar mais sobre o funcionamento dos vetores em linguagens de programação. Bons Estudos!
